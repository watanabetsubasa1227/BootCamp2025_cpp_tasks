## 概要
これは2025年のBootCamp C++編の課題です．
C++の基礎的な使い方に慣れ，後の課題の足がけになる知識を身に着けましょう．

具体的には，標準入出力，関数，クラス，スレッド，シリアル通信の基礎的な使い方を会得してもらいます．


## 課題の進め方
課題はTodoの内容を補っていく形になっています．

課題内容を確認して一つ一つ進めていきましょう！

ChatGPTやGemini等のAIを使っても構いません．手も足も出ないよりかは，AIを使いながらでも完成させた方が価値があると思います．

しかし，AIを使って答えた場合は必ず仕組みをAIに質問して理解するように務めてください．

AIは何回同じ質問をしても嫌な顔をせずに答えてくれます．うまく活用してください！
